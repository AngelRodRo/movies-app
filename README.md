## 🚀 Getting Started

```
npm install
npm run start
```

The `npm run start` command execute for default in port `3000`.
Also, you can check deployed app on: https://hardcore-bassi-24e03c.netlify.app/

if you can test components isolated run:

```
npm run storybook
```
